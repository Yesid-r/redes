HTTP: Protocolo de Transferencia de Hipertexto-Hypertext Transfer Protocol

Puerto TCP 80

Es el protocolo de comunicación que se utiliza para transferir datos a través de Internet, especialmente en la World Wide Web. HTTP funciona mediante una solicitud-respuesta entre un cliente y un servidor web. Cuando un usuario introduce una URL en un navegador web, el navegador envía una solicitud HTTP al servidor web para obtener la página web correspondiente. El servidor web procesa la solicitud y envía una respuesta HTTP al navegador, que contiene la página web solicitada.
